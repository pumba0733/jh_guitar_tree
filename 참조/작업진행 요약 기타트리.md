## 🛠️ 현재까지 수정된 주요 파일 목록

- `login_screen.dart`: 로그인 방식 개선 (동명이인 대응)
- `firestore_service.dart`: Firestore 연동, 요약 저장 메서드 추가
- `ai_service.dart`: Gemini 프롬프트 생성 + 응답 처리
- `lesson_summary_screen.dart`: 요약 요청 UI 구성
- `summary_result_screen.dart`: 요약 결과 확인용 화면
- `local_hive_boxes.dart`: Hive 박스 초기화 구성
- `today_lesson_screen.dart`: 수업 주제, 음원, 메모 입력 UI 구현
